## CSS form
``` CSS
@font-face{
font-family: [string];
font-style: normal;
font-weight: 300 (Light) | 400 (Regular) | 500 (Medium) | 600(Semiblod) | 700 (Bold) | 800(Dark | Heavy);
src: url("[url]") format("eot"),
	url("[url]") format("embedded-opentype"),
	url("[url]") format("woff"),
	url("[url]") format("tff");
}
```
